Жаңа!  Psiphon 3 құралын сынап көріңіз

Psiphon 3 деген не?

Psiphon 3 – Psiphon Inc. компаниясы ұсынған ақаулықтарды айналып өтуге арналған, VPN, SSH және HTTP прокси технологиясын қолданатын, мүмкін болған ең жақсы кіру рұқсатын қамтамасыз ететін жаңа құрал. Сіздің Psiphon 3 клиентіңіз ақаулықтарды айналып өту мүмкіндігін арттыру үшін автоматты түрде жаңа кіру нүктелері туралы білетін болады.

Psiphon 3 құралына қалай қол жеткіземін?

Клиент бағдарламасын жүктеп алу үшін мына  жүктеп алу сілтемесін  басыңыз.

Қазіргі уақытта Psiphon 3 құралы Windows XP/Vista/7 жүйелерін талап етеді. Mac OS X және Android қосқандағы мобильді платформаларына қолдау көрсетуі жақын арада күтіледі!

Psiphon 3 – ашық бастапқы жоба. Бастапқы код пен дизайн құжаттарын  жобаның бастапқы бетінен таба аласыз

Psiphon 3 құралын қалай іске қосамын?

Клиент бағдарламасын жүктеп алып, іске қосыңыз. Іске қосқан уақытта бұл бағдарламаның Psiphon Inc заңды өнімі екендігі туралы қауіпсіздік ескертпесі көрінуі тиіс.

Psiphon 3 құралын іске қосқан уақытта автоматты түрде сервермен байланыс орнатуын бастайды. Байланысу уақытында айналмалы белгіше көрінеді. Сіз келесідей туннель режимдерін таңдай аласыз: VPN (L2TP қатысты IPSec), SSH немесе SSH+ (SSH плюс – протокол саусақ іздеріне жол бермейтін SSH төбесіндегі кездейсоқ қабат).

Жасыл белгіше жанған уақытта, сіздің интернет трафигіңіз Psiphon 3 арқылы жүреді. VPN режимінде сіздің барлық трафигіңіз автоматты түрде Psiphon 3 арқылы өтеді.

SSH және SSH+ режимдерінде, Psiphon 3 автоматты түрде Windows жүйесінің прокси параметрлерін орнатады және осы параметрлерге тәуелді қолданбаларға арналған трафик Psiphon 3 арқылы өтеді. Бұл параметрлер көптеген веб шолғыштары тарапынан әдепкі ретінде орнатылған.

Сонымен қатар, SSH және SSH+ режимдерінде, Psiphon 3 халықаралық трафик прокси арқылы өтіп, ішкі трафик өтпейтін бөлетін туннель опциясын ұсынады. Бөлетін туннельден өткізуді іске қосу үшін “Проксиден өткізбеу...” опциясын тексеріңіз. Бұл опция қосулы тұрғанда проксиден өтпеген домендер хабарлама аймағында көрсетіледі.

Бағдарламаны жапқан уақытта, Psiphon 3 автоматты түрде ажырайды. Байланысты ауыстырып қосу үшін белгішені де баса аласыз.
