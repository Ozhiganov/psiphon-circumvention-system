Что такое Psiphon 3?

Psiphon 3 - это новое средство преодоления интернет цензуры от Psiphon Inc., которое использует технологии VPN, SSH и HTTP прокси для решения проблемы доступа к заблокированным ресурсам. Программный клиент Psiphon 3 автоматически 'узнает' о новых серверах и, таким образом, увеличивает Ваши шансы обхода цензуры.

Psiphon 3 предназначен, чтобы предоставить Вам открытый доступ к онлайн-контенту. Использование Psiphon не приводит к увеличению вашей конфиденциальности в Интернете, и не следует рассматривать или использовать его в качестве инструмента онлайн безопасности.

Является ли мой Psiphon 3 для Windows подлинным?

Дистрибутив Psiphon 3 для Windows не является инсталляционным пакетом. Каждый программный клиент Psiphon 3 для Windows является исполняемым приложением(".exe") с цифровой подписью Psiphon Inc. Windows автоматически проверяет эту подпись при запуске клиента. Вы также можете вручную проверить подпись перед запуском клиента, вызвав диалоговое окно Свойств файла и проверив вкладку Цифровые Подписи.

Цифровой отпечаток SHA1 открытого ключа сертификата Psiphon Inc. можно проверить во вкладке Подробности диалога Сертификат. Для сертификата, срок действия которого с 16 Июня 2011 до 21 Июня 2012 отпечаток SHA1: 

        8f:b7:ef:bd:20:a9:20:3a:38:37:08:a2:1e:0a:1d:2e:ad:7b:ee:6d
и для сертификата, срок действия которого с 21 Мая 2011 - 30 Июля 2014 отпечаток SHA1: 
        84:c5:13:5b:13:d1:53:96:7e:88:c9:13:86:0e:83:ee:ef:48:8e:91
Psiphon 3 для Windows обновляется и проверяет подлинность каждое обновления автоматически.

Как пользоваться Psiphon 3 для Windows?

Скачайте программный клиент и запустите его. После запуска вы должны увидеть диалог предупреждения безопасности показывающий, что программа является официальным продуктом Psiphon Inc.

Psiphon 3 автоматически инициирует соединение с сервером при запуске. В процессе соединения видна вращающаяся иконка. Вы можете выбрать один из следующих режимов туннелирования: VPN (L2TP/IPSec), SSH, or SSH+ (обфусцированный SSH, чтобы избежать идентификации протокола).

Зеленая иконка означает, что соединение с сервером Psiphon установлено.В режиме VPN весь ваш интернет-трафик идет через Psiphon 3.

В режимах SSH и SSH+ Psiphon 3 автоматически устанавливает системные настройки прокси Windows и трафик приложений, которые соблюдают эти настройки идет через Psiphon 3. Эти настройки соблюдаются по умолчанию всеми основными современными веб-браузерами.

Кроме того, в SSH и SSH + режимов Psiphon 3 предлагает режим выборочного туннелирования при котором международный интернет-трафик идет через прокси, а внутренний - нет. Установите флажок у опции “Don’t proxy...”, чтобы включить режим выборочного туннелирования. Когда эта опция включена, вы можете видеть домены, трафик к которым не идет через Psiphon 3 в окне сообщений.

Psiphon 3 автоматически прекращает соединение, когда вы закрываете программу. Вы также можете щелкнуть по иконке, чтобы начать/прекратить соединение.

Как пользоваться Psiphon 3 для Android?

Нажмите на ссылку Psiphon APK из вашего клиента электронной почты Android или браузера, чтобы начать установку.

Для установки Psiphon APK, опция "Неизвестные Источники" в меню настроек приложений в вашем Android устройстве должна быть включена. Psiphon 3 для Android будет автоматически обновлять себя.

При запуске приложения Psiphon 3, оно автоматически попытается инициализировать подключение к сети Psiphon.

Как только приложение подключится к сети Psiphon, запустится встроенный Psiphon браузер. Psiphon 3 для Android не туннелирует трафик встроенного браузера Android или других приложений. По умолчанию туннелируется лишь трафик встроенного браузера Psiphon 3.

Какую информацию о пользователях собирает Psiphon 3?

Мы собираем следующие данные, чтобы выяснить, насколько хорошо Psiphon работает, какие сайты являются популярными, и какие стратегии распространения приложения являются эффективными. Мы делимся этой информацией с нашими партнерами, чтобы они могли видеть, например, как часто их сайты посещают через Psiphon и из каких стран.

  • Количество запросов на электронную почту для получения ссылки для скачивания
  • Количество апгрейдов
  • Как часто каждый протокол используется, и коды ошибок сбоя
  • Частота обнаружения новых серверов
  • Количество пользовательских сессий и их продолжительность
  • Количество просмотров (полные адреса страниц сайтов спонсоров)
  • Количество переданных байт
  • Количество запросов HTTPS
  • Клиентская платформа (используя упрощенный перечень операционных систем, а не, например, детальный User Agent браузера)

IP-адреса и адреса электронной почты пользователей не собираются и пользователи не должны создавать учетные записи для использования системы.

Логи событий включают временные метки, коды региона(GeoIP код страны) и не идентифицирующие атрибуты, включая ID спонсора (определяется тем, какой клиент Psiphon используется), версия клиента, а также тип протокола. Количество просмотров агрегируется по времени и / или сессии перед записью на диск.

Вся статистика доступная для спонсоров далее агрегирована по дате, спонсору и региону.

Время от времени Psiphon, возможно, придется записывать дополнительную информацию для того, чтобы решить проблему с нашим сервисом. Если это произойдет, мы добавим здесь анонс с описанием того, что было записано, как долго длилась запись и почему.

Другие наиболее часто задаваемые вопросы

Q: Где я могу получить последнюю информацию о Psiphon 3?

Q: Почему IP адреса Psiphon 3 так часто меняются?

A: Ваш клиент Psiphon 3 будет автоматически 'узнавать' о новых Psiphon 3 серверах. Если последний использованный сервер в настоящее время недоступен, то Psiphon произведет попытку соединения со следующим сервером из списка.

Q: Почему я вижу постоянно повторяющееся сообщение "connection failed"(сбой подключения)?

A: Если вы видите повторяющееся сообщение "connection failed"(сбой подключения), то это означает, что ваш клиент не 'знает' доступных в данное время серверов. Попробуйте скачать новый клиент Psiphon 3.

Q: Где скачать новый Psiphon 3?

A: Отправьте электронное письмо по адресу get@psiphon3.com или по адресу который вы использовали для получения текущей версии вашего Psiphon 3 приложения.

Q: Как мне проверить версию Psiphon 3?

A: Когда Psiphon 3 стартует, версия клиента отображается в окне сообщений.

Q: Что за файл "psiphon3.exe.orig"?

A: Процесс автоматического обновления в Psiphon 3 для Windows переименовывает старую версию в "psiphon3.exe.orig". Старые файлы с расширением ".orig" могут быть безопасно удалены.

Q: Туннелирует ли Psiphon 3 для Windows весь мой интернет-трафик?

A: Только в режиме VPN. После успешного установления соединения в режиме VPN весь интернет-трафик вашего компьютера будет идти через сеть Psiphon 3. Только те приложения, интернет соединение которых может быть настроено через HTTP и SOCKS прокси могут использовать Psiphon 3 туннель в режиме SSH.

Q: Совместим ли Psiphon 3 для Windows с IE, Firefox, Safari и Chrome?

A: Да. При работе в режимах SSH проверьте настройки вашего браузера и убедитесь, что он настроен на использование системных настроек прокси.

Q: Существуют ли какие-либо ограничения портов в режиме VPN? Почему я не могу посылать электронную почту, используя почтовый клиент в режиме VPN?

A: Исходящие соединения с Psiphon 3 для Windows в режиме VPN могут быть установлены только на следующих портах: 53, 80, 443, 554, 1935, 7070, 8000, 8001, 6971-6999. Смотреть обсуждение для получения дополнительной информации. Почтовые клиенты не могут устанавливать исходящие соединения на портах 25 и 587. Смотреть обсуждение для получения дополнительной информации.

Q: Какой VPN протокол используется Psiphon 3 для Windows? Почему я не могу подключиться?

Psiphon использует VPN протокол L2TP/IPSec. Возможно брандмауэр вашей сети не позволяет использование VPN. Возможно ваш домашний маршрутизатор настроен на запрет этого VPN протокола; проверьте настройки маршрутизатора и убедитесь, что сквозной(passthrough) режим для IPSec или L2TP включен. Также, возможно IPSec сервис вашей системы отключен, проверьте настройки и включите автоматический запуск этого сервиса.

Q: Я могу подключиться к Psiphon 3 для Windows в режиме VPN, но скорость соединения очень мала. Иногда веб-страницы не загружаются вообще. Почему?

A: Некоторые модели сетевого оборудования или интернет-соединения могут создавать проблемы с производительностью протокола L2TP/IPSec, который используется Psiphon 3 в режиме VPN. Попробуйте использовать SSH режим.

Q: При подключении к Psiphon 3 для Windows в режиме VPN, ни одна из моих веб-страниц не грузится. Я получаю сообщения об ошибках о том, что Интернет домен не найден.

A: Psiphon 3 ограничивает DNS трафик 'белым списком' проверенных DNS серверов. Клиент Psiphon 3 автоматически устанавливает настройки DNS VPN. Если вы получаете ошибки, связанные с DNS, убедитесь, что Ваша система не заражена вредоносными программами "DNS Changer", которые пытаются изменить DNS настройки. Более подробная информация тут

Q: Как настроить приложения для использования Psiphon 3 туннеля в режимах SSH?

A: Psiphon 3 автоматически установит системные настройки Windows на использование локальных прокси серверов - HTTP / HTTPS прокси на 127.0.0.1:8080 и SOCKS прокси на 127.0.0.1:1080. Windows приложения, которые используют прокси настройки системы будет автоматически использовать туннель. Вы можете вручную настроить другие приложения на использование этих локальных прокси-серверов. Эти прокси сервисы доступны как в Psiphon 3 для Windows (SSH режимы), так и в Psiphon 3 для Android.

Q: Что такое режим SSH+?

A: SSH+ это режим SSH с обфускацией на уровне инициализации SSH протокола между клиентом и сервером для защиты от профилирования типа протокола. Описание протокола может быть найдено тут

Q: В SSH режиме Psiphon 3 для Windows я вижу сообщение об ошибке "Failed to connect to Polipo". Что делать?

A: Локальный прокси-сервер HTTP не запустился. Возможно существует другой процесс, который уже занял этот порт. Попробуйте использовать другой порт. См. вопрос "Могу ли я использовать локальный прокси-сервер HTTP на отличном от 8080 порту?"

Q: Могу ли я использовать локальный прокси-сервер HTTP на отличном от 8080 порту?

A: Да, на Windows. Нажмите Run и введите "regedit", чтобы открыть редактор реестра. Найдите и откройте "HKEY_CURRENT_USER \ Software \ Psiphon3" и вы увидите переменную "UserLocalHTTPProxyPort". Установите ее значение в то, которые вы хотели бы использовать для порта HTTP прокси.

Q: Я пользуюсь AutoProxy. Как запретить Psiphon 3 для Windows менять системные настройки прокси?

A: Нажмите Run и введите "regedit", чтобы открыть редактор реестра. Найдите и откройте "HKEY_CURRENT_USER \ Software \ Psiphon3" и вы увидите переменную "UserSkipProxySettings". Установите ее значение равным 1 и Psiphon 3 не станет изменять системные настройки прокси при последующих запусках.
