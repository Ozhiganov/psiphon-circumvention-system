Psiphon 3 бу нима?

Psiphon 3 бу Psiphon Inc. компанияси томонидан яратилган VPN ва HTTP Proxy технологияларни ишлатиб интернет маълумотига кириш мумкин бўлган айланма йўлни берадиган янги воситадир. Сизнинг Psiphon 3 мижозингиз интернет маълумотига киришнинг янги жойлардан автоматик тарзда ҳабардор бўлиб, сизни назоратдан четлаб ўтиш имконларингизни кучайтиради.

Psiphon 3 ни қаердан топишим мумкин?

Psiphon 3 ни қандай ишга тушираман?

Мижозлар дастурини юклаб олинг ва ишга туширинг. Уни ишга туширганингизда, мазкур дарстур Psiphon Inc. компаниясининг қонуний маҳсулоти эканлигини кўрсатадиган ҳимоя кўрсатмасини кўришингиз керак.

Уни ишга туширганингизда Psiphon 3 автоматик тарзда уланишни бошлайди. Уланилаётган вақтда айланадиган белги пайдо бўлади.

Яшил белги пайдо бўлганда сизнинг Интернет трафикингиз Psiphon 3 орқали ўтяпти.

Дастурни ёпганингизда, Psiphon 3 автоматик тарзда алоқадан узилади. Уланишни ўзгартириш учун иконачага ҳам босишингиз мумкин.

Айрим вақтларда VPN протоколи беркиланганда, 20 сониядан сўнг уланиш бўлмаса, Psiphon 3 автоматик тарзда қайтарилиб SSH туннели орқали HTTP Proxy ва SOCKS Proxy-ни таклиф қилади. Мазкур тартибда Интернет трафикингизнинг ҳаммаси ҳам Psiphon 3 орқали ўтмайди, фақат маълум иловалар ўтади.
