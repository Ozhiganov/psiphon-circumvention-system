Янги!  Psiphon 3 ни ишлатиб кўринг

Psiphon 3 нима?

Psiphon 3 бу Psiphon Inc. компанияси томонидан яратилган VPN ва HTTP прокси серверлар ёрдамида Интернетга айланма йўл билан кириш имконини берадиган янги воситадир. Psiphon 3 клиенти автоматик тарзда Интернетга киришнинг янги айланма йўлларини топиб, қўйилган тўсиқларни четлаб ўтиш имконини оширади.

Psiphon 3 ни қаердан топишим мумкин?

Psiphon 3 клиент дастурини юклаб олиш учун мана  бу линкни  босинг.

Psiphon 3 дастурини ишлатиш учун Windows XP/Vista/7 бўлиши керак. Mac OS X ва мобил платформалар, жумладан Android учун Psiphon 3 тез орада ишлаб чиқарилади!

Psiphon 3 ни очиқ манбалардан бепул олиш мумкин. Манба коди ва унга оид ҳужжатларни  project home page дан топишингиз мумкин.

Psiphon 3 ни қандай ўрнатаман?

Psiphon 3 клиент дастурини компьютерга юклаб олиб, ўрнатинг. Дастурни ўрнатганда, унинг Psiphon Inc. компаниясига доир қонуний маҳсулоти эканлигини англатувчи ёзув пайдо бўлади.

Дастур ўрнатилиши биланоқ Psiphon 3 автоматик тарзда Интернетга уланади. Уланаётганини англатувчи икона пайдо бўлади.

Яшил икона Интернет трафик Psiphon 3 орқали ўтаётганини англатади.

Дастурни ёпганингизда, Psiphon 3 автоматик тарзда алоқадан узилади. Алоқа каналини ўзгартириш учун тегишли белгини босишингиз мумкин.

Баъзан VPN протоколи тўсилган бўлиши мумкин, бу ҳолда 20 секунд ўтгач, Psiphon 3 автоматик тарзда SSH канали орқали HTTP ва SOCKS прокси серверлар орқали кириш йўлларини таклиф қилади. Бу режимда Интернет трафикнинг ҳаммаси Psiphon 3 орқали ўтмайди, айрим иловалар ўтади холос. Psiphon 3 Windows тизимидаги прокси тартибини автоматик равишда барча оммабоп браузерлар тан оладиган режимга мослайди.
