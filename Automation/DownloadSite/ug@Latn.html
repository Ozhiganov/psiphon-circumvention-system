Yéngi!  Sayfon 3 ni sinap béqing

Sayfon 3 déginimiz néme?

Sayfon 3 sayfon shirkiti teripidin yézip chiqilghan muweqqet mulazémitir yumtali bolup, u mewhum mexsus tor (VPN) we HTTP muweqqet mulazémitiri arqiliq sizni torgha kirishtiki eng yaxshi usul bilen teminleydu. Sayfon 3 yumtali aptomatik bilish usuli arqiliq, yéngi kirish nuqtilirini bayqap, sizning tosulghan torlargha kirishingizning mumkinchilikini ashuridu.

Sayfon 3 ke qandaq érishimen?

Bu  yüklesh  ulinishini bésip, yumshaq détalni köchürüng.

Nöwette, Sayfon 3 Windows XP/Vista/7 sharaitida ishleydu. Mac OS X we Android qol téléfonlirini öz ichige alghan yanfon nusxiliri yéqinda élan qilinidu.

Sayfon 3 menbe kodi ochuq projékt bolup, uning menbe kodi we layihilinish matériyallirini  projékt torbéti  din tapalaysiz.

Sayfon 3 ni qandaq ishlitimen?

Yumtalni yüklep, qozghisingizla boldi. Uni qozghatqiningizda bir bixeterlik uchuri chiqip, bu yumtalning Sayfon shirkiti (Psiphon Inc) teripidin ishlepchiqirilghanliqini eskertidu.

Sayfon 3 qozghitilghanda aptomatik halda torgha ulinishni bashlaydu. Uliniwatqanda siz bir pirqirawatqan belgini körisiz.

Yéshil renglik belge körün'gende, sizning tordiki paaliyetliringhiz Sayfon 3 arqiliq bolidu.

Siz programmini taqighanda, Sayfon 3 aptomatik halda üzülidu. Siz yéshil renglik belgini bésish arqiliqmu tor ulinishini bashquralaysiz.

Bezide, mewhum mexsus tor (VPN) tosulup qalghanda, 20 sékonttin kéyin Sayfon 3 aptomatik halda SHH tonéli arqiliq HTTP muweqqet mulazémitiri we SOCKS muweqqet mulazémitirigha ulinidu. Bu xil usulda sizning tordiki paaliyetliringhizning hemmisila Sayfon 3 arqiliq bolmaydu, Sayfon 3 Windows séstimisining muweqqet mulazémitirliq artuqchiliqidin paydilinidu. Bu köpinche asasliq tor körgüchiliri qollaydighan usuldur.
