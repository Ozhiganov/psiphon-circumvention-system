Yangi!  Psiphon 3 ni ishlatib ko'ring

Psiphon 3 nima?

Psiphon 3 bu Psiphon Inc. kompaniyasi tomonidan yaratilgan VPN va HTTP proksi serverlar yordamida Internetga aylanma yo'l bilan kirish imkonini beradigan yangi vositadir. Psiphon 3 klienti(yuzeri) avtomatik tarzda Internetga kirishning yangi aylanma yo'llarini topib, qo'yilgan to'siqlarni chetlab o'tish imkonini oshiradi.

Psiphon 3 ni qayerdan topishim mumkin?

Psiphon 3 klient (yuzer) dasturini yuklab olish uchun mana  bu linkni  bosing.

Psiphon 3 dasturini ishlatish uchun Windows XP/Vista/7 bo'lishi kerak. Mac OS X va mobil platformalar, jumladan Android uchun Psiphon 3 tez orada ishlab chiqariladi!

Psiphon 3 ni ochiq manbalardan bepul olish mumkin. Manba kodi va unga oid hujjatlarni  project home page dan topishingiz mumkin.

Psiphon 3 ni qanday o'rnataman?

Psiphon 3 klient(yuzer) dasturini komputerga yuklab olib, o'rnating. Dasturni o'rnatganda, uning Psiphon Inc. kompaniyasiga doir qonuniy mahsuloti ekanligini anglatuvchi yozuv paydo bo'ladi.

Dastur o'rnatilishi bilanoq Psiphon 3 avtomatik tarzda Internetga ulanadi. Ulanayotganini anglatuvchi ikona (belgi) paydo bo'ladi.

Yashil ikona(belgi) Internet trafik Psiphon 3 orqali o'tayotganini anglatadi.

Dasturni yopganingizda, Psiphon 3 avtomatik tarzda aloqadan uziladi. Aloqa kanalini o'zgartirish uchun tegishli belgini bosishingiz mumkin.

Ba'zan VPN protokoli to'silgan bo'lishi mumkin, bu holda 20 sekund o'tgach, Psiphon 3 avtomatik tarzda SSH kanali orqali HTTP va SOCKS proksi serverlar orqali kirish yo'llarini taklif qiladi. Bu rejimda Internet trafikning hammasi Psiphon 3 orqali o'tmaydi, ayrim ilovalar o'tadi xolos. Psiphon 3 Windows tizimidagi proksi tartibini avtomatik ravishda barcha ommabop brauzerlar tan oladigan rejimga moslaydi.
