Yangi!  Psiphon 3ni ishlatib ko'ring

Psiphon 3 bu nima?

Psiphon 3 bu Psiphon Inc. kompaniyasi tomonidan yaratilgan VPN va HTTP Proxy tyexnologiyalarni ishlatib intyernyet ma'lumotiga kirish mumkin bo'lgan aylanma yo'lni byeradigan yangi vositadir. Sizning Psiphon 3 mijozingiz intyernyet ma'lumotiga kirishning yangi joylardan avtomatik tarzda habardor bo'lib, sizni nazoratdan chyetlab o'tish imkonlaringizni kuchaytiradi.

Psiphon 3 ni qayerdan topishim mumkin?

Mijozlar dasturini yuklab olish uchun mazkur  download link  ulanish tugmasini bosing.

Hozirgi vaqtda Psiphon 3 uchun Windows XP/Vista/7 dasturi kyerakdir. Mac OS X va mobil platformalar, jumladan Android uchun dasturlar tyez orada ishlab chiqariladi!

Psiphon 3 bu byepul ommabop dasturdir. Siz kod manbasini va loyiha hujjatlarini   project home page -dan topishingiz mumkin.

Psiphon 3 ni qanday ishga tushiraman?

Mijozlar dasturini yuklab oling va ishga tushiring. Uni ishga tushirganingizda, mazkur darstur Psiphon Inc. kompaniyasining qonuniy mahsuloti ekanligini ko'rsatadigan himoya ko'rsatmasini ko'rishingiz kyerak.

Uni ishga tushirganingizda Psiphon 3 avtomatik tarzda ulanishni boshlaydi. Ulanilayotgan vaqtda aylanadigan byelgi paydo bo'ladi.

Yashil byelgi paydo bo'lganda sizning Intyernyet trafikingiz Psiphon 3 orqali o'tyapti.

Dasturni yopganingizda, Psiphon 3 avtomatik tarzda aloqadan uziladi. Ulanishni o'zgartirish uchun ikonachaga ham bosishingiz mumkin.

Ayrim vaqtlarda VPN protokoli byerkilanganda, 20 soniyadan so'ng ulanish bo'lmasa, Psiphon 3 avtomatik tarzda qaytarilib SSH tunnyeli orqali HTTP Proxy va SOCKS Proxy-ni taklif qiladi. Mazkur tartibda Intyernyet trafikingizning hammasi ham Psiphon 3 orqali o'tmaydi, faqat ma'lum ilovalar o'tadi. Psiphon3 asosiy Intyernyet ko'rib chiquvchi dasturlar tomonidan o'rnatilgan tartibda qabul qilinadigan Windows tizimining syervyer o'rnatmalarini avtomatik tarzda ulaydi.
